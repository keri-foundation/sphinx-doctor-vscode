# Roadmap

## Phase 0

Documentation bootstrap.

Deliverables:

- repo purpose
- product scope
- target architecture
- research intake surface

## Phase 1

Contract and fixtures.

Deliverables:

- versioned schema
- representative raw and enriched fixtures
- documented mirror layout
- documented multi-root settings contract
- documented AI packet shape

## Phase 2

Enrichment engine.

Deliverables:

- one reliable path for converting inventory issues into enriched diagnostics
- source-range and confidence metadata
- honest retention of unmapped issues

## Phase 3

Minimal diagnostics extension.

Deliverables:

- load enriched diagnostics
- publish and clear native VS Code diagnostics
- keep extension state file-backed rather than hidden

## Phase 4

Tree View and AI packet refinement.

Deliverables:

- grouped navigation and stable AI packets
- trusted command execution only after read-only flows are proven
- code actions much later and only for narrow cases
